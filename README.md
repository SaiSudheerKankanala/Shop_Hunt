The ShopHunt application was designed with a strong backend architecture that integrates SQL-based inventory management and a GenAI-powered chatbot interface. I built the backend using structured database design principles, creating a unified SQL table (product_inventory) to store complete product, shop, and stock-related information. The database schema includes indexed fields for optimized search queries, automated timestamps, and dynamic stock status updates using enums and boolean fields.

The application backend handles CRUD operations, manages real-time inventory updates, and ensures efficient data retrieval for product search, price lookup, and availability checks. Additionally, I integrated a Generative AI chatbot that communicates with the backend to respond to user queries such as product details, shop availability, pricing, quantity, and expiry/manufacture dates. The chatbot uses natural language understanding to convert user queries into backend-friendly operations, improving the overall user experience.

Overall, the project demonstrates backend development, database schema design, SQL query optimization, and GenAI chatbot integration to build a smooth and intelligent product-search platform.
